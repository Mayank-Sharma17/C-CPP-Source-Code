Q1 : What is tokens in C ?

In C programming, tokens are the smallest individual units of a program that are meaningful to the compiler. They are the fundamental building blocks from which all C programs are constructed. Think of them like the words and punctuation marks that form sentences in a natural language; without them, the compiler cannot understand the program's structure or intent.

There are six main types of tokens in C:

Keywords: These are reserved words in C that have predefined meanings and cannot be used for any other purpose, such as variable names. Examples include int, if, while, for, return, void, etc.

Identifiers: These are names given by the programmer to various program elements like variables, functions, arrays, structures, etc. They must follow specific naming rules (e.g., start with a letter or underscore, contain letters, digits, or underscores). Examples include main, totalSum, myVariable.

Constants: These are fixed values that do not change during the execution of a program. They can be of various types: 
Integer constants: 10, 0x2A (hexadecimal), 021 (octal)
Floating-point constants: 3.14, 2.5f
Character constants: 'a', '\\n' (newline character)
String literals: "Hello, World!"

Operators: These are symbols that perform operations on operands (variables or constants). Examples include:
Arithmetic operators: +, -, *, /, %
Relational operators: ==, !=, <, >, <=, >=
Logical operators: &&, ||, !
Assignment operator: =

Special Symbols (Punctuators): These are symbols that have specific structural or organizational roles in the C language. Examples include:
Parentheses () (for function calls, expressions)
Curly braces {} (for code blocks)
Square brackets [] (for arrays)
Comma , (to separate items)
Semicolon ; (to terminate statements)
Hash # (for preprocessor directives)

Strings: These are sequences of characters enclosed in double quotes, representing text. While technically a type of constant (string literal), they are often treated as a distinct category due to their frequent use and specific handling. Example: "This is a string."
